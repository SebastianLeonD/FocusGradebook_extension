How to Use Focus Grade Simulator

Basic Usage

  1. Select either Unweighted or Weighted calculator based on your class grading system
  2. Enter the Points Earned you'd get on a hypothetical assignment
  3. Enter the Points Possible for that assignment
  4. Select a Category (required for weighted grades)
  5. Click Add Assignment to see your updated grade

Special Cases

Zero-Point Assignments

If you have an assignment that has a 0:

  1. Enter what you would have scored in the Points Earned field
  2. Enter 0 in the Points Possible field
  3. This will correctly calculate as extra credit

Note: This is useful for assignments that count as extra credit or for overriding an assignment that was marked as zero.

Tips

  • Use Reset All to clear all hypothetical assignments
  • The extension works across multiple classes - each class tracks separate hypothetical assignments
  • Your hypothetical grade appears in red at the top of your gradebook, or at the blue hypothetical box.
